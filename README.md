# alpine-simh








simh built from source.

To run the included machines change directory to the machine directory and run the command:

```Example

cd /machines/altairz80
altairz80 cpm3 

```

```
i:a:r ZORK1.COM
a:r ZORK1.DAT
a:r ZORK2.COM
a:r ZORK2.DAT
a:r ZORK3.COM
a:r ZORK3.DAT
```
